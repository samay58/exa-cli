# Roadmap

## Near-Term

- dedicated `company` command built on Exa company search conventions
- dedicated `people` command with stricter domain and filter validation
- terminal-safe device-flow auth
- richer `doctor` diagnostics for hosted MCP clients
- signed release automation for Homebrew and release tarballs
- optional live smoke coverage for research create/cancel workflows

## Medium-Term

- optional shell completion packaging in release artifacts
- richer LLM output shaping for multi-query coding workflows
- deeper local history browsing for prior runs and research tasks
- better human formatting for company and people results

## Deliberately Not In V1

- npm and PyPI packaging
- background daemons
- hidden browser auth assumptions
- auto-updaters that mutate local state on their own
- kitchen-sink command sprawl
